MCQQ Bridge bundled NapCat

Release packages include NapCat.Shell.Windows.OneKey from NapNeko/NapCatQQ.
Source code does not vendor NapCat binaries or QQ runtime files.

Default OneBot endpoints:

HTTP: http://127.0.0.1:3000
WS:   ws://127.0.0.1:3001

First launch still requires QQ login or QR scan in the NapCat/QQ window.
